# Live Debug Session

## Session purpose

Fast one-bug-at-a-time debugging while the user manually tests the app on a real device/emulator.

## Rules

- no PR per bug
- no commit per bug
- no push per bug
- no broad refactor per bug
- one bug at a time
- minimal fix
- focused check
- user verifies
- batch verification after bug group
