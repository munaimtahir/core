# Decision 0003 — Baseline First

## Status

Accepted.

## Decision

The project will build a stable launcher baseline before adding product-specific features.

## Reason

Specialized features depend on stable launcher behavior.

## Consequences

The baseline excludes:

- caregiver features
- parent/child features
- school/office features
- remote/cloud features
- premium features
- kiosk/device-owner features

These may be planned later after baseline GO.
