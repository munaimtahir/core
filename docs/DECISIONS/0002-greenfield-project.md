# Decision 0002 — Greenfield Project

## Status

Accepted.

## Decision

`core` is a brand-new greenfield project.

It must not reuse legacy code, architecture, workflows, UI flows, or state systems from any existing project.

## Reason

The foundation should be designed specifically for the current baseline scope.

## Consequences

- New architecture.
- New implementation.
- New tests.
- New CI setup.
- New documentation.
- No copied legacy source files.
- No inherited legacy state systems.
