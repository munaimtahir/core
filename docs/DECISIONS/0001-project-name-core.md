# Decision 0001 — Project Name: core

## Status

Accepted.

## Decision

The internal repository name is:

```text
core
```

The internal project title is:

```text
Core Launcher Foundation
```

## Reason

The project is intended to become a shared foundation for a future launcher family.

A neutral internal name keeps the foundation separate from any one product variant.

## Consequences

- `core` is not the public product name.
- Future products can have separate branding.
- The baseline remains product-neutral.
